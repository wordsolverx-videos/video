# Wordsolver Video Sync

This directory contains the logic for synchronizing Quordle solve video links from the solver to the frontend repository.

## Contents
- `update_repo.py`: The main script that handles cloning/pulling the video repository and creating daily JSON files.
- `instructions.txt`: Setup instructions for local and automated usage.
- `github_actions_guide.md`: Detailed guide on setting up the GitHub Actions workflow and secrets.
